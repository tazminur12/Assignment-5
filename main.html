The different ways to select an element in the DOM are: 1. By ID - document.getElementById("id") 2. By Class Name - document.getElementsByClassName("class") 3. By Tag Name - document.getElementsByTagName("tag") 4. By Name Attribute - document.getElementsByName("name") 5. By CSS Selector - 1. Single element: document.querySelector("selector") 2. Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML: Returns or sets the HTML content inside an element, including tags. innerText: Returns or sets only the visible text inside an element, ignoring hidden elements and formatting. textContent: Returns or sets all text inside an element, including hidden text, but without parsing HTML tags.
Question-3: What is event delegation in the DOM?
Event delegation in the DOM is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single listener is added to their common ancestor, improving performance and simplifying event management.
Question-4: What is event bubbling in the DOM?
Event bubbling is a DOM event propagation method where an event starts from the target element and moves up to its ancestors. For example, if you click a button inside a "div", the event first triggers on the button, then bubbles up to the "div", and then to higher ancestors like "body". This allows event delegation and efficient event handling.
Question-5: How do you create, add, and remove elements using JavaScript?
1. Create an element: document.createElement("tagName"), 2. Add an element- Append to a parent: parent.appendChild(child), Insert before another element: parent.insertBefore(newElement, referenceElement), 3. Remove an element: element.remove(), parent.removeChild(child).
